What is scope? and how does it works in JavaScript?

Scope Put simply: When a variable is declared outside a function it has a global scope, which means it can be passed in to any function in your code. When it is declared inside a function it is known as having local scope, meaning it is only available locally within that function. for example: var globalVariable=();
