What a responsive site is, and why responsiveness is important?

A Responsive site is a site that still looks good when seen on an iPhone, an iPad or a desktop computer. Before responsive sites, everything would stay the same size, but the window would get smaller and you’d have to scroll left to right to see it properly. Responsive design works so that everything on the site will shrink with the window size and will still look good.


What mobile first design is, and why it's important?

Mobile first design is when you design for mobile first, as it says, you start designing making sure the mobile view looks great first, then you work up from there, to tablet, and then desktop. It is important so instead of the design degrading as the size gets smaller, it rather expands and still looks good as the screen gets bigger.


What frameworks are, and their pros and cons?

Frameworks are pre written CSS files that you integrate into your webpage, so you don’t have to write every single element out. The Pro of this is that they save you time as most f the groundwork is done, and you make your own personal changes. The Con is that if there is an element you want to change, its a bit more difficult and you have to go into the bones of the framework to change it.


What a wireframe is and why we use it.

A wireframe is a mock up of a site, it is either sketched out by hand, or there are fancy programs that can draw them up, either way is acceptable, it is just a guide for you to plot out the look and layout of the website.
