Single or double quotes main purpose is to define strings. Javascript can read both double and single quotes as the same, however, if you are using them with a string that contains quote punctuation within it, then you need to use double quotes: example "you can't do this without double quotes" because if it were 'you can't do this wihtout double quotes' the computer would read the can't quote as the end!
